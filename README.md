# KITTI_VIZ_3D
Visualization 3D object detection results using meshlab.

This code is used for visualization by adding 3D bounding boxes to the LiDAR point cloud and storing it in a txt file.

## Requrements

numba

opencv

matplotlib

pandas


## python code

kitti_util.py

meshlab_file.py

draw3Dbox2img.py
